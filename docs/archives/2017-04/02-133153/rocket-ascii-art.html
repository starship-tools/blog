Rocket ASCII Art

Sometimes plain text just says it best ...

April 02, 2017

by oubiwann

A quick internet search for "ascii rocket" turned up the following little gem:

                       *     .--.
                            / /  `
           +               | |
                  '         \ \__,
              *          +   '--'  *
                  +   /\
     +              .'  '.   *
            *      /======\      +
                  ;:.  _   ;
                  |:. (_)  |
                  |:.  _   |
        +         |:. (_)  |          *
                  ;:.      ;
                .' \:.    / `.
               / .-'':._.'`-. \
               |/    /||\    \|
         jgs _..--"""````"""--.._
       _.-'``                    ``'-._
     -'                                '-

JGS, whoever you are, that's one of the most delightful bits of ASCII art I've seen. This makes me want to write a children's space adventure book, illustrated only with ASCII art ...

More availble here:

  • http://textart.io/art/tag/spaceship/1

Don't be surprised if these start showing up in code samples ;-)

Update: A message on Twitter indicated that JGS is Joan Stark. Thanks, @thatgrrl!
